enum liballuris_error

liballuris specific errors

Enumerator
LIBALLURIS_SUCCESS 

No error.

LIBALLURIS_MALFORMED_REPLY 

The received reply contains a malformed header. This should never happen, check EMI and physical connection

LIBALLURIS_DEVICE_BUSY 

Device is in a state where it cannot process the request. The device cannot perform the requested operation. Some examples:

  • get digits or units while measurement is running
  • get or change upper or lower limit while measurement is running
  • USB API not available, for example on a FMI-S10 device
LIBALLURIS_TIMEOUT 

No response or status change in given time

LIBALLURIS_OUT_OF_RANGE 

Parameter out of valid range or invalid mode/state For example set unit which isn't supported (for example 'oz' for 500N device)

struct alluris_device_description *  alluris_devs,
size_t  length,
char  read_serial 
)

List accessible alluris devices.

The product field in alluris_device_description is filled via the USB descriptor. After this the device is opened and the serial_number is read from the device. Thus this function only lists devices where the application has sufficient rights to open and read from the device. Check permissions if a device isn't returned.

The retrieved list has to be freed with liballuris_free_device_list before the application exits.

int liballuris_get_digits ( libusb_device_handle *  dev_handle,
int *  v 
)

Query the number of digits for the interpretation of the raw fixed-point numbers.

All liballuris_get_* functions return fixed-point numbers or strings. This function queries the number of digits after the radix point. For example if liballuris_get_value returns 123 and digits returns 1 the real value is 12.3. Use liballuris_get_unit to get a meaningful measurement.

Query this value is only possible if the measurement is not running, else LIBALLURIS_DEVICE_BUSY is returned.

Parameters
[in]dev_handlea handle for the device to communicate with
[out]voutput location for the returned number of digits. Only populated if the return code is 0.
Returns
0 if successful else liballuris_error
